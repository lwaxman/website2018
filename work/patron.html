The Patrón Experience

Educating customers on Patrón tequila in augmented reality.

user experience, visual design, 3d

In collaboration with M Booth, Midnight Commercial created an augmented reality application for Patrón using Apple’s freshly released ARKit. The application brings a virtual miniature of the Patrón Hacienda to the user’s tabletop, complete with agave field and jimadors, 4 virtual bottles of Patrón tequila and an animated bartender to tell the user about each expression.

I was responsible for the user experience and design, from onboarding to the AR enhanced image capture feature.
